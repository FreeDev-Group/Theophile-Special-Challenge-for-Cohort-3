## Testing Guide for Contributors
Welcome to the testing phase of the Student Survey App. To ensure a smooth testing process, please follow these instructions:

## Getting Started
Before you begin testing, it is essential to understand how the application is designed to behave:

- Understand the Logic: Navigate to the /login folder and review the Use Case document. This will provide you with the necessary context on how the application should function.

## Reporting Your Tests
To maintain consistency across all reports, please use our standardized format:

- Test Template: Refer to the template_for_test.md file.

- Instructions: Use this Markdown template as a guide to document your testing process, including expected results, actual outcomes, and any bugs encountered.

## Your Mission
Your goal is to verify that the application aligns perfectly with the provided Use Cases. If you find any discrepancies, please document them using the template provided.
